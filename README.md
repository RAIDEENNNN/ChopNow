# ChopNow

A reusable, dependency-free restaurant storefront with menu filtering, a persistent cart, delivery pricing, and WhatsApp checkout.

## Preview

Open `index.html` directly, or serve this folder from any static host.

## Customize for a customer

Edit the `STORE` and `MENU` values at the top of `app.js`. Change the colours in `:root` at the top of `styles.css`, then update the name and text in `index.html`.

Before delivery, replace the placeholder WhatsApp number (`2348000000000`) with the restaurant's number in international format without `+`.

All names, prices, delivery promises, ratings, and customer counts are demonstration content. Replace them with facts supplied by the restaurant before publishing.
